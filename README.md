# Website_with_templates
 
## Вебсайт с шаблонами из публичного доступа

Сайт, в котором был разработан бэкенд, а фронтенд часть была представлена в виде бесплатных шаблонов на бирже шаблонов для вебсайтов.
The site where the backend frontend part was developed was presented in the form of free templates on the website template exchange.
